## Scope

Admin manages their tenant company:
- Driver lifecycle (applications, credentials, status)
- Vehicle management and assignments
- Broker relationships and requirements
- Trip manifests and driver pay
- Rate configuration
- Team members (coordinators)
- Company settings

## Roles

- **Admin**: Full access to all features
- **Coordinator**: Limited access (see permission matrix in 03-AUTHENTICATION.md)
