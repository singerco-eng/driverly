## Scope

Drivers manage their own:
- Profile and personal information
- Vehicles (if 1099 / own vehicle)
- Credentials and documentation
- Availability schedule
- Trip acceptance/rejection
- View payments and earnings
- Communication with admin

## Driver States

1. **Pending** - Application submitted, awaiting review
2. **Under Review** - Admin actively reviewing
3. **Approved** - Can access full portal
4. **Rejected** - Application declined
5. **Active** - All credentials valid, can take trips
6. **Inactive** - Missing credentials or admin-deactivated
